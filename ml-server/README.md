# ML Take Home - backend

FastAPI Backend for the ML Take Home project.

## Quickstart

```bash
# Clone the codebase
git clone git@github.com:Monadical-SAS/ML-take-home.git
cd ML-take-home

# Init the server
cd ml-server 
pipenv install
pipenv shell
python3 main.py 
```

To run the tests
```shell
# Running the tests
pytest
```
